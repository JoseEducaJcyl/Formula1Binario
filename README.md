# Gestor de Pilotos F1 - Versión Modular con Serialización

## Descripción
Aplicación modular en Java para gestionar pilotos de Fórmula 1. Combina serialización de objetos con operaciones de gestión (añadir, buscar, mostrar) y persistencia en archivos binarios.

## Funcionalidades
- Crear pilotos con nombre, escudería y puntos
- Generar datos de ejemplo automáticamente
- Guardar lista de pilotos en archivo binario
- Leer pilotos desde archivo binario
- Añadir nuevos pilotos a la lista
- Buscar pilotos por nombre
- Mostrar todos los pilotos registrados

## Tecnologías
- Java
- Serialización (interfaz Serializable)
- ObjectOutputStream / ObjectInputStream
- ArrayList
- Clases y métodos estáticos
- Scanner para entrada de usuario
- Manejo de excepciones
